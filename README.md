# Linear Regression From Scratch

This project demonstrates an implementation of **Linear Regression from scratch**
using Python, without relying on machine learning libraries like scikit-learn.

## Dataset
- Ecommerce Customers Dataset

## Features
- Data loading and preprocessing
- Cost function implementation
- Gradient Descent optimization
- Model training 

## Requirements
- Python 3.x
- NumPy
- Pandas
- matplotlib

## How to Run

python linear_regression_from_scratch.py
